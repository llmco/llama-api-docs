# Obtaining an API Token for LLM Api

## Step 1: Access our Website

To access our website, you must first Create an Account. To do this,
visit [https://www.llmapi.com](https://www.llmapi.com), click on
_Log In_ --> _Sign up_ and follow the steps on the screen.

<img height="200" src="/images/signup-screen.png" alt="signup" />

## Step 2: Waitlist

Llm is currently in a Private Beta; so, when you signup, you are added to our waitlist.
Once you receive our invitation, you can follow the next steps on this Doc to become one
of our first customers!

<Info>
  Usually we are able to approve your profile in a few days, so keep an eye out
  for our email!
</Info>

### Step 3: Obtain an API Token

Once your registration is complete and your account has been approved, log in and navigate to [API Token](https://www.llmapi.com/account/api-token).

<img height="200" src="/images/login.png" alt="token button" />

Hover over the clipboard icon and copy your token.

<img height="200" src="/images/token-api.png" alt="token" />

Now, you are ready to be one of the first testers of LLM Api! Follow the [Quick Start Guide](/quickstart) for more information.
